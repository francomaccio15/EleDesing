⚡ Cómo Usar los Agentes

📍 Ubicación

Todos los agentes están instalados globalmente y disponibles en todos tus proyectos de ChatBots:

~/.claude/agents/
🗣️ Activación por Texto

Simplemente pedíselo en el chat al inicio de una conversación:

Actúa como el engineering-devops-automator
Quiero que uses el rol de sales-proposal-strategist
Comportate como el support-analytics-reporter
🔀 Pipeline Multi-Agente

Para proyectos complejos, activá primero el orquestador:

Actúa como el agents-orchestrator para este proyecto

El orquestador coordinará automáticamente los demás agentes necesarios (architect → developer → reviewer → tester).

💡 Casos de Uso Reales para Axis

Deploy CI/CD: @engineering-devops-automator → "Diseñame un pipeline GitHub Actions para el VPS Hostinger con PM2"

Propuesta de cliente: @sales-proposal-strategist → "Necesito una propuesta para la Municipalidad de Tartagal"

Reporte ejecutivo: @support-executive-summary-generator → "Generá un reporte mensual para TravelRock con estos datos"

Review de código: @engineering-code-reviewer → [pegar código] → "Revisá este endpoint de autenticación"
